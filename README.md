# Winter Olympics Medal Analysis

## Overview
This project analyzes Winter Olympic medal data with a focus on identifying long-term patterns, performance drivers, and structural differences between countries, sports, and demographics.

The project combines Python-based data preparation with interactive Power BI dashboards designed for exploratory analysis and insight generation.

## Data
- Olympic medal data (Year, Sport, Event, Country, Medal, Athlete/Team, Age)
- Country ranking data
- Public sources: Wikipedia, Sports-Reference, LA Times

## Tools
- Python (pandas)
- Power BI (DAX)
- GitHub

- ## Contents
- **PDF report** – static overview for quick review
- **Power BI (.pbix)** – interactive dashboard
- **Python notebook** – data preparation and feature engineering

## Key Analytical Questions
- Which countries dominate Winter Olympics historically?
- How has medal distribution evolved over time?
- What is the age profile of medal-winning athletes?
- Are there structural differences by gender or sport?

## Deliverables
- Power BI dashboard (interactive)
- Python notebook for data preparation
- Exported PDF report with key visualizations
